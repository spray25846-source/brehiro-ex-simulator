# ブレヒロ EX成長シミュレーター（検証用）

本ツールは、ブレイブフロンティアヒーローズ（ブレヒロ）における  
EXレベルによるステータス上昇を  
**実測データをもとに検証・再現するためのシミュレーター**です。

## 対応レアリティについて

現在、本シミュレーターは  
**アンコモンレアリティのユニットのみ**を対象としています。

これは、アンコモンユニットについてのみ

- EX0〜EX20までの実測データ
- 成長傾向（0〜12メモリ）を網羅した検証データ

が揃っているためです。

レア、エピック、レジェンダリー以上のレアリティについては  
現時点では対応していません。

## 今後の対応予定について

レアリティごとのEX成長仕様については、  
アンコモン以外のレアリティでも同様の法則が成り立つ可能性がありますが、

- EX強化に必要なコストが非常に高いこと
- 成長傾向を全パターン検証するためのデータ収集負荷が大きいこと

といった理由から、  
**現時点ではアンコモン以外のレアリティへの対応は未定**としています。

将来的に十分な検証データが集まった場合には、  
対応レアリティを拡張する可能性もありますが、  
必ずしも実装を保証するものではありません。

## 注意事項

- 本シミュレーターは検証・研究目的のツールです  
- ゲーム内の最終的な数値や挙動を **100%保証するものではありません**
- 実際のゲーム仕様と差異が生じる可能性があります

---

## セキュリティ・プライバシーについて

- 本アプリはサーバー側にユーザーデータを保存しません
- 入力された数値は計算処理のみに使用されます
- アカウント登録・個人情報の入力は不要です
